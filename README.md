
# NLP XML Text Processing Project

This project loads text from XML files, cleans it, applies NLP techniques, and exports results for analysis.

Features

Extract text from XML files

Clean & preprocess text

Named Entity Recognition (NER)

TF-IDF vectorization

Bag-of-Words (BoW)

WordCloud generation

Sentiment analysis

Machine Learning models (SVM & Logistic Regression)

Export results to CSV & Excel

Technologies Used

Python

BeautifulSoup

ElementTree (XML parsing)

NLTK

spaCy

Scikit-learn

WordCloud

TextBlob

Pandas

How to Run

Install required libraries:

pip install -r requirements.txt
python -m spacy download en_core_web_sm


Place XML files in your project folder.

Run the Python script or Jupyter notebook.

Output files (CSV, Excel, WordCloud) will be created automatically.
